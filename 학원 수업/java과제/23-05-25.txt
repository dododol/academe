메소드의 헤드부분
: 접근지정자 제한자 반환타입 메소드명(매개변수)

접근지정자 : public, protected, default, private
제한자 : abstract, final, static
반환타입 : 데이터타입, void

인터페이스

: - 클래스가 아니다
  - 추상메소드, 정적상수만 포함될 수 있다
  - 일반변수는 올 수 없다. 일반변수가 오게되면 자동으로 public static final로 변경된다.

추상클래스의목적과 겹치는 부분이 많다.
-가이드라인을 제공. 

인터페이스 = 인터페이스로 상속. 다중상속 가능
interface A extends B,C,D

ERROR	컴파일 - 문법
	실행 - 예외처리대상

예외처리 : 정상종료로 유도.

예외는 사용자의 잘못된 조작, 코딩으로 발생하는 프로그램 오류

예외처리 프로그램 통해 정상 실행상태 유지 가능

exception = 가장 촘촘한 그물. 못잡는 에러가 없다.
